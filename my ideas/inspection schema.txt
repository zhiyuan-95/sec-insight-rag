# Phase 1: XBRL Data Engineering

In this part of project, we will be able to ingest data related to financial fact from xbrl to our local machine, so that which can be easily retrieved for later usage. 

company requested → check if exists → initialize if missing → ingest XBRL → store locally → check update only when needed

## 1.1 If the requested company is not in the system, run initialization.

USE XBRL API, get corresponding company info

### Initialization:

**create object tables companies, ingest financial data from sec XBRL to local sql**

- **TABLE → companies**:
    
    **company_id:** internal UUID or integer, AUTOINCREMENT
    **cik:** SEC CIK, canonical ID,
    **name:** current SEC company name,
    **ticker**: primary ticker of this company
    **sic**: 4-digit industry code,
    **sic_description:** corresponding industry of the sic code
    **fiscal_year:** 2026
    **last_10k_filing_date:** filing date of the last ingested 10k data
    **last_10q_filing_date:** filing data of the last ingested 10q data
    **next_check_date_10k:** generated by the function below, 
    **next_check_date_10q:** generated by the function below,
    **created_at:** first time we initialize the company object
    **updated_at:** last time we do update
    
    (we don’t have 10k, 10q, we use last_10k_filing_date refers to refer the latest annule report and it is provided on xbrl, as well as last_10q_filing_date)
    
    update_check_start_date_10k = previous_business_day(last_10K_filing_date + 1 year)
    
    If latest 10-Q is Q1 or Q2:
    update_check_start_date_10q = previous_business_day(last_10Q_filing_date + 3 months)
    
    If latest 10-Q is Q3:
    update_check_start_date_10q = previous_business_day(last_10Q_filing_date + 6 months)
    

```sql
CREATE TABLE companies (
    company_id INTEGER PRIMARY KEY AUTOINCREMENT,
    cik TEXT UNIQUE NOT NULL,
    name TEXT NOT NULL,
    ticker TEXT,
    exchange TEXT,
    sic TEXT,
    sic_description TEXT,
    fiscal_year TEXT,
    last_10k_filing_date date,
		last_10q_filing_date date,
		next_check_date_10k date,
		next_check_date_10q date,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

```sql
CREATE TABLE filings (
filing_id INTEGER PRIMARY KEY AUTOINCREMENT,
company_id INTEGER NOT NULL,
accession_number TEXT UNIQUE NOT NULL,
form_type TEXT NOT NULL,          -- 10-K, 10-Q, 10-K/A, 10-Q/A
filing_date DATE NOT NULL,
report_date DATE,
fiscal_year INTEGER,
fiscal_period TEXT,               -- FY, Q1, Q2, Q3
source TEXT DEFAULT 'SEC',
ingested_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
FOREIGN KEY (company_id) REFERENCES companies(company_id)
);
```

## 1.2 If the requested company already exists in the system, check whether updates are needed.

For each form group: 10-K and 10-Q

1. If today < update_check_start_date:
    
    Do not check XBRL.
    Retrieve existing data from the database.
    
2. If today >= next_check_date_10k or next_check_date_10q:
    
    Query xbrl website for all filings where:
    form_type is 10-K or 10-Q
    filing_date > last_processed_filing_date
    
    1. If no new filings are found:
        
        next_check_date remain the same
        Retrieve existing data from the database.
        
    2. If one new filing is found:
        
        Ingest the new filing.
        Update last_processed_filing_date.
        Generate a new next_check_date_10k/10q base on the filing_date of the file ingested.
        
    3. If multiple new filings are found:
        
        Ingest all missing filings in chronological order.
        Update last_processed_filing_date to the newest filing.
        Generate a new next_check_date_10k/10q base on the filing_date of the last file ingested.
        

## Extract and Ingest

**Extract following facts:**

**Income statement**: revenue, cost_of_revenue, gross_profit, operating_income, net_income, eps_basic, eps_diluted, r_and_d_expense, sga_expense, interest_expense, income_tax_expense

**Balance sheet:** cash_and_equivalents, accounts_receivable, inventory, total_assets, short_term_debt, long_term_debt, total_debt, accounts_payable, total_liabilities, shareholders_equity

**Cash flow:** operating_cash_flow, capital_expenditure, free_cash_flow, investing_cash_flow, financing_cash_flow, share_repurchases, dividends_paid

**Ingest into following table**

- **TABLE → financial_facts**

```sql
CREATE TABLE financial_metrics (
    metric_id INTEGER PRIMARY KEY AUTOINCREMENT,
    company_id INTEGER NOT NULL,
    statement_type TEXT NOT NULL, -- income_statement, balance_sheet, cash_flow
    metric_name TEXT NOT NULL,    -- revenue, net_income, assets, cash, etc.
    value REAL,
    unit TEXT,
    period_type TEXT NOT NULL,    -- annual, quarterly, ttm, instant
    fiscal_year INTEGER,
    fiscal_period TEXT,
    start_date DATE,
    end_date DATE,
    filing_date DATE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(company_id, metric_name, period_type, fiscal_year, fiscal_period, accession_number),
    FOREIGN KEY (company_id) REFERENCES companies(company_id)
);
```
